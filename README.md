# Loan Data Visualization

This project explores and visualizes a loan dataset using Python in a Jupyter Notebook. 
The goal is to analyze loan patterns and generate insights through data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib 
- Jupyter Notebook

## Dataset
The dataset (`loan_data.csv`) contains loan-related information used for exploratory data analysis and visualization.

## Objectives
- Load and explore loan data
- Clean and prepare the dataset
- Create visualizations to understand trends and patterns
- Generate insights from the data

## Files
- `Untitled.ipynb` – Jupyter notebook containing the analysis and visualizations
- `loan_data.csv` – dataset used in the analysis
